## Getting Started

### Dependencies

* Angular-cli, Node, npm

### Installing
```
$> npm install
```

### Executing
* Localhost server
```
$> ng serve
```

* Unit test (no-watch)
```
$> npm run test
```

* E2E test (no-watch)
```
$> npm run e2e
```
